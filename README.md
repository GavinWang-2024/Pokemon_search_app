## About The Project

A application that allows users to search for Pokémon by name or ID and display information about them.

### Features

- **Pokémon Search**: Search for Pokémon by name or ID.
- **Detailed Information**: View detailed information including weight, height, types, and base stats.
- **Responsive Design**: Optimized for different screen sizes.
